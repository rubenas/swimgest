Installation instructions

1.- Enable extension GD in php.ini and restart the server in order img treatment works. 
2.- Create a database.
3.- Add an user to this new database with privileges on it.
4.- Upload all files in a folder on your server and go to yourdomain/folder
5.- Follow installation steps. You will need smtp credentials for mailing service.
6.- An super-admin user is created with unername admin@admin.com and pass '1234'. You have to change this password on first login. This user won't bee able to be deleted by anyone.
7.- Now you can login and start to add swimmers, events...
8.- Enjoy :)
